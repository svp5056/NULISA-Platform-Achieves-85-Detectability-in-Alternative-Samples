Alamar Biosciences has published performance data for its NULISA proteomics platform across 10+ alternative sample types, demonstrating attomolar sensitivity and a 12-log dynamic range. Validated detectability includes CSF (85%+), dried blood spots (80–95%), interstitial fluid (95%+), saliva (75–85%), urine (45–75%), nasal swabs (70–80%), and tear fluid (60–75%). Experiments were performed on the automated ARGO HT System with up to 288 samples per run and less than 30 minutes hands-on time.

Source: https://alamarbio.com/ultra-sensitive-proteomics-alternative-sample-types/
