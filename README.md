# NULISA-Platform-Achieves-85-Detectability-in-Alternative-Samples
Alamar Biosciences published data showing the NULISA proteomics platform across 10+ alternative sample types with attomolar sensitivity and a 12-log dynamic range. Validated detectability includes CSF (85%+), DBS (80–95%), saliva (75–85%), urine (45–75%), and more using the automated ARGO HT System.
